#### a が b と等しい
```python
a = 1
b = 1

# True
print(a == b)
```

#### a が b と異なる
```python
a = 1
b = 1

# False
print(a != b)
```

#### a が b よりも小さい
```python
a = 1
b = 1

# False
print(a < b)
```

#### a が b よりも大きい
```python
a = 1
b = 1

# False
print(a > b)
```

#### a が b 以下である
```python
a = 1
b = 1

# True
print(a <= b)
```

#### a が b 以上である
```python
a = 1
b = 1

# True
print(a >= b)
```

#### a も b も真であれば真
```python
a = 1
b = 1

# True
print(a > 0 and b > 0)
```

#### a または b が真であれば真
```python
a = 1
b = 1

# True
print(a > 0 or b > 0)
```
